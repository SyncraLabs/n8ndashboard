# Syncra Web 1 - AI Automation Dashboard

## Objetivo
Crear un dashboard de rendimiento de una automatización de IA con datos inventados.

## Requisitos de Diseño
- **Estilo**: Minimalista y muy bonito.
- **Branding**: Blanco, limpio, muy luminoso.
- **Acentos**: Gradientes verdes.
- **Animaciones**: Super chulas, fluidas (usando Framer Motion).

## Componentes Planeados
1. **Header/Navbar**: Simple, con el logo (texto) y tal vez un avatar de usuario.
2. **Key Metrics (KPIs)**: Tarjetas mostrando métricas clave (ej. Tareas automatizadas, Tiempo ahorrado, Precisión de IA, Tasa de éxito).
3. **Charts**: 
   - Gráfico de área para el rendimiento a lo largo del tiempo.
   - Gráfico de barras para el tipo de tareas automatizadas.
4. **Recent Activity**: Lista de las últimas acciones ejecutadas por la IA, con estados (Éxito, Fallo, Pendiente).
5. **System Status**: Un pequeño widget mostrando que el sistema está "Online" y el uso de recursos.

## Stack
- Vite + React + TypeScript
- Tailwind CSS v4 (para estilos y utilidades)
- Framer Motion (para animaciones)
- Recharts (para gráficos)
- Lucide React (para iconos)
